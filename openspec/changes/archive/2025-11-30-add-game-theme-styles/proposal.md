# 变更: 添加游戏风格/主题设置功能

## 为什么

当前 Word Cannon 游戏只有单一的视觉风格（蓝色渐变主题）。添加主题自定义功能可以让用户根据个人喜好选择不同的视觉风格，如赛博朋克、机器猫或蝙蝠侠主题，从而提高用户参与度，让学习体验更加有趣。

## 变更内容

- 在设置页面添加新的"风格设置"区域
- 实现基于 CSS 变量的主题系统，方便切换主题
- 创建 4 种独特的主题：
  1. **默认** - 当前的蓝色渐变主题
  2. **赛博朋克** - 霓虹灯色彩、深色背景、科技感
  3. **机器猫** - 蓝白配色、圆角元素、可爱风格
  4. **蝙蝠侠** - 暗黑风格、黑黄配色、哥特氛围
- 主题选择保存到 localStorage，持久化存储
- 切换主题时无需刷新页面，动态应用

## 影响范围

- **新增规格**: `game-themes` 游戏主题功能
- **涉及代码**:
  - `proj/css/styles.css` - 添加 CSS 变量，替换硬编码颜色
  - `proj/css/settings.css` - 主题选择界面样式
  - `proj/index.html` - 应用主题类名
  - `proj/settings.html` - 主题选择界面
  - `proj/src/ui/settings.js` - 主题切换逻辑
  - 新文件: `proj/css/themes.css` - 主题定义
