# Change: 添加考级系统（Level Certification System）

## Why

目前游戏只有练习模式，缺乏系统化的学习进度追踪和成就激励机制。学生需要一个考级系统来：
1. 明确学习路径（音标 → 年级 → 进阶分支）
2. 获得阶段性成就感（徽章奖励）
3. 建立长期学习目标
4. 提供多元化发展路径（课外阅读 / 升学考试）

## What Changes

### 新增功能

1. **考级系统核心**
   - **四大徽章系列**，采用树状解锁结构：
     - 音标系列（前置，始终解锁）
     - 小学年级系列（需音标徽章解锁）
     - 课外阅读系列（需6年级徽章解锁，分支1）
     - 升学系列（需6年级徽章解锁，分支2）
   - 通过标准：正确率 ≥ 90%
   - 逐级解锁机制

2. **级别结构**
   - **音标系列**：5个小级别（短元音、长元音、双元音、辅音、复合音）→ 1枚徽章
   - **小学年级系列**：4个大级别（3-6年级），每级2个小级别 → 4枚徽章
   - **课外阅读系列**：4个大级别 → 4枚徽章
     - Fly Guy（苍蝇小子）：小级别待定
     - Magic Tree House（神奇树屋）：小级别待定
     - Dragon Ball（七龙珠）：小级别待定
     - Harry Potter（哈利波特）：小级别待定
   - **升学系列**：3个大级别 → 3枚徽章
     - 初中（7-9年级）：小级别待定
     - 高中（高一至高三）：小级别待定
     - 四级（CET-4）：小级别待定

3. **徽章系统**
   - 欧洲古老家族徽章风格
   - 大级别徽章：共 **12枚**（音标1 + 小学4 + 课外阅读4 + 升学3）
   - 小级别用分数/进度表示，不设徽章
   - 未通过：灰色徽章；通过后：彩色点亮
   - 通过大级别后可悬挂徽章

4. **主页徽章区**
   - 位置：主页面左上角
   - 显示已获得的徽章
   - 无徽章时显示挑衅手势
   - 点击进入考级页面

5. **考级页面**
   - 类似游戏升级加点模式的 UI
   - **树状技能树布局**：
     ```
     音标 → 3年级 → 4年级 → 5年级 → 6年级 ─┬→ 课外阅读分支
                                            └→ 升学分支
     ```
   - 徽章悬挂展示区
   - 级别进度和分数显示

## Impact

- Affected specs: 新增 `level-certification` 能力规范
- Affected code: 
  - `proj/index.html` - 主页徽章区
  - `proj/src/` - 新增考级系统模块
  - `proj/css/` - 徽章和考级页面样式
  - `proj/words/` - 可能需要按音标类型重组词库
  - `proj/words/extracurricular-books/` - 课外阅读词库整合
